RUCT
This is a simple expositive Github project site dedicated to RUCT - a growing repository which consist of scripts and programs that helps you with managing your containers.
The problem that coused the intention seems to be simple to understand. Containerization is quickly becoming a trend which causes it to be used even when it is not supposed to.
For that use cases those scripts may be really helpful.
